# 极速车库 - 3D 跳一跳游戏

一个基于 HTML5 和 Three.js 的 3D 跳跃游戏，用于营销推广活动。

Demo：https://file.348580.xyz/2025/12/c431339b7bee3d6f68550279ba1e6945.html

- 建议使用手机浏览器打开体验

## 游戏玩法

1. 长按屏幕蓄力，控制跳跃力度
2. 跳跃到下一个平台
3. 落在中心区域获得「完美」+2 分，普通区域 +1 分
4. 掉落后可用复活卡或关注公众号复活
5. 达到指定分数可解锁租车优惠券

## 技术栈

- HTML5 + CSS3 + JavaScript
- Three.js (r128) - 3D 渲染
- 纯前端实现，无后端依赖

## 运行方式

直接在浏览器中打开 `jump_jump.html` 即可运行游戏。

## 游戏预览

- **角色**: 白色方块小人（带蓝色耳朵和眼睛）
- **平台**: 不同车型的租车权益区（经济型轿车、特斯拉、极氪001、保时捷911 等）
- **特效**: 完美跳跃光环动画、蓄力挤压变形、掉落动画

## 奖励机制

| 分数 | 奖励 |
|------|------|
| 50分 | 10元优惠券 |
| 100分 | 9折租车券 |
| 200分 | 60元次卡 |
